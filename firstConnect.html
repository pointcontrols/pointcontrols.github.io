Connecting to a PC using Wi-Fi

You can download the pc app from the internet or directly from this phone

Downloading from the internet

You can download the pc application from mpget.me Then please read this

Downloading pc app directly from phone

Make sure your phone and PC are connected to the same Wi-Fi network. Please restart the app if you had to change the network connection. Open windows explorer on your PC. Click on the tab on the left. You should see this entry in the "Other devices" group:

If you don't see it, try refreshing by pressing the button to the left of the search box. If the device is still missing, please take a look at Enabling network discovery

Double click on the "MotionPoint app" entry. A browser will open with a request to download a zip file.

Running the pc application

Please save and extract the archive. Run the PointControl application. If you have enabled firewall, a dialog will ask you to allow access through it.

You can now close this help window, and wait for your PC to appear in the scan results. You can open this help again from the options menu.

Connecting to a PC using Bluetooth

You can download the pc app from mpget.me. Please save and extract the archive. Ensure bluetooth is enabled on your PC. You should see the bluetooth icon in your system tray. Then run the PointControl application. If not already on, please enable bluetooth discovery by clicking on the bluetooth slider.

On your phone, if Bluetooth is off please enable it and restart the app. In the connect screen, click on the button named "Bluetooth". A scan will take place, which may take several minutes. Once you see your pc listed, click on it.
